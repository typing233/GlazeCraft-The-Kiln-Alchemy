# 🏺 GlazeCraft: 窑变炼金术

> 结合3D物理模拟与AIGC技术的Web端互动数字陶艺创作与传统文化沉浸式科普平台

## 📖 项目简介

GlazeCraft 是一款创新的数字陶艺创作平台，让玩家能够在Web浏览器中体验完整的传统陶瓷制作流程。从3D拉坯塑形，到古籍配方解谜，再到窑变烧制，每一个环节都经过精心设计，让玩家在创作中感受中国传统陶瓷文化的魅力。

### ✨ 核心功能

- **🎨 3D实时拉坯塑形** - 基于Three.js的物理模拟，鼠标实时交互
- **📜 古籍配方解谜** - 三种经典釉料配方（青瓷、钧窑、建盏），精准调配
- **🔥 窑炉烧制系统** - 可控制温度曲线、氧气含量、冷却速率
- **🎭 AIGC窑变生成** - 基于Perlin噪声和物理参数的程序化纹理生成
- **📚 个人数字图鉴** - 作品保存、管理、展示
- **🔗 社交分享功能** - 一键生成3D展示链接

## 🛠️ 技术栈

### 后端
- **Python 3.8+**
- **Flask 3.0** - Web框架
- **Flask-CORS** - 跨域处理
- **SQLite** - 数据存储
- **Pillow (PIL)** - 图像处理
- **NumPy** - 科学计算

### 前端
- **HTML5 / CSS3 / JavaScript ES6+**
- **Three.js r128** - 3D渲染引擎
- **OrbitControls** - 相机控制

## 🎮 功能使用指南

### 第一阶段：拉坯塑形

1. 点击 **"启动转轮"** 按钮开始旋转
2. 选择合适的工具：
   - **拉坯**: 向外扩展器壁
   - **推压**: 向内收缩器壁
   - **捏塑**: 塑造口沿细节
   - **光滑**: 平整表面
3. 调整 **转轮速度** 控制旋转快慢
4. 鼠标左键在3D模型上拖动来塑造形状
5. 完成后点击 **"下一步: 调配釉料"**

### 第二阶段：釉料配方

1. 从左侧选择一个配方：
   - **青瓷釉** (简单): 宋代龙泉窑经典釉色
   - **钧窑釉** (困难): 入窑一色，出窑万彩
   - **建盏油滴釉** (专家): 兔毫纹与油滴结晶的艺术

2. 在右侧调整原料比例，带 `*` 标记的为必选原料
3. 点击 **"验证配方"** 检查比例是否正确
4. 实时预览釉色效果
5. 验证通过后点击 **"下一步: 入窑烧制"**

### 第三阶段：烧制过程

1. 调整烧制参数：
   - **升温速率**: 控制窑炉升温快慢
   - **氧气含量**: 决定氧化焰/还原焰 (>15%氧化, <10%还原)
   - **冷却速率**: 影响结晶形成

2. 烧制流程：
   - 点击 **"开始烧制"** 升温
   - 达到目标温度后点击 **"保温"**
   - 点击 **"开始冷却"** 降温
   - 或直接点击 **"紧急停止"** 中断

3. 实时查看：
   - 温度数值显示
   - 温度曲线图表
   - 烧制阶段状态

4. 冷却完成后点击 **"完成烧制"**

### 第四阶段：作品完成

1. 查看窑变统计：
   - 最高温度
   - 氧气含量
   - 冷却速率
   - 窑变等级评定

2. 为作品命名并添加描述
3. 点击 **"保存到图鉴"** 保存作品

### 第五阶段：数字图鉴

1. 查看所有保存的作品
2. 点击作品卡片查看详细信息
3. 点击 **"生成分享链接"** 创建可分享的3D展示页面
4. 复制链接分享到社交网络

## 🎨 窑变算法说明

### 核心原理

系统采用程序化生成算法，根据以下参数决定最终釉色效果：

1. **原料成分**
   - 氧化铁(Fe₂O₃): 影响青色/褐色调
   - 氧化铜(CuO): 影响绿色/蓝色调
   - 五氧化二磷(P₂O₅): 促进结晶形成
   - 草木灰: 增加透明度和流动性

2. **烧制参数**
   - **最高温度**: 影响釉料熔融程度
   - **氧气含量**: 决定氧化还原状态
     - 氧化焰(>15%): 铁呈三价，显褐色
     - 还原焰(<10%): 铁呈二价，显青色
   - **冷却速率**: 影响结晶形成
     - 快速冷却: 形成更多细小结晶
     - 缓慢冷却: 形成大尺寸结晶

3. **Perlin噪声**
   - 多层噪声叠加产生自然纹理
   - 随机种子确保每件作品独一无二

### 窑变等级评定

| 等级 | 条件 |
|------|------|
| 极品窑变 | 最高温度>1250°C + 氧气<15% + 冷却速率>50°C/分 |
| 优质窑变 | 最高温度>1200°C + (氧气<18% 或 冷却速率>30°C/分) |
| 正常烧成 | 最高温度>1100°C |
| 普通 | 其他情况 |
